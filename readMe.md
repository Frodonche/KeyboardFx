Il s'agit du projet de test pour JavaFX
Au final, le clavier contient seulement le pav. num. (aucun intérêt de faire le reste)

Features :
- Affichage d'un pav. num. avec qqes couleurs basiques
- Changement de couleur symbolisant l'appui de la touche

Note :
- Normalement JavaFX est inclut dans les JDK de java 8, ce n'étais pas le cas sur mon Ubuntu
Solution : sudo apt-get install openjfx
